# Hide polito courses from web portal

I don't know ehy they do not provide this feature by default like in the app

# Step to configure it

1. Clone the repository
   ```
   git clone https://github.com/salvatorecalo/disable_polito_courses.git
   ```
3. Open a browser and digit chrome://extensions (i didn't test it for other browser since i don't use them)
   *if you want to port it to them make me PR*
5. click upload extension and upload it
   <img width="3022" height="1356" alt="image" src="https://github.com/user-attachments/assets/e6ad3c5f-de20-438b-b899-ce21742d4597" />
7. go to polito portal and test it
<img width="3006" height="1690" alt="image" src="https://github.com/user-attachments/assets/813c6ec2-2a3a-4f17-96b1-7f474ca07624" />

Click on the course you want to disable
